# Ron's Actuarial Portfolio

Actuarial projects built using Philippine data, developed while pursuing
ASP credentials and transitioning into the actuarial profession.

All analyses use publicly available data from the Philippine Statistics
Authority (PSA) and international open datasets where local data is unavailable.
